# GAME CENTER APP## Project overview: * Project created to learn Java and Spring.* View layer served on server side with Thymeleaf.* Database – PostgreSQL with Flyway version control.## Application overview:* Application to play Battleships online against other players.Key features:  ####GAME PAGE:1. Two grids; to place your own ships and to strike opponent ships.2. Chat window to show both players moves and to communicate with opponent.3. Possibility to pause game, once player wish to resume, mail will be sent to other one asking to join.####GAME LOBBY:1. Leaderboard2. Created games list where you could join.##Project phases:1. [STARTED]  | 01.2022 | Models, DB, Basic views.
